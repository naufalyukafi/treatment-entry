# Treatment Entry Service

## Stack
- Expres.js 
- Firebase Firestore (Firebase Admin)
  
## System Requirement

Recommended dependencies are :
- node v20.9.0
- yarn v1.22.21

## Configuration file

Add file serviceAccountKey.json to root server folder

## Run 
Run for development / local 
```
cd existing_repo
yarn install
yarn run dev
```
